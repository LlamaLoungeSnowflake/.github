# Barack Ollama

Save yourself the time to tailor your resume and reorganize your GitHub profile and your portfolio page for every job posting you apply for.
Barack Ollama automates and personalizes all of that witha  few simple clicks. Under the hood there are checks and reasoning to get the most factual and optimal outcome.
